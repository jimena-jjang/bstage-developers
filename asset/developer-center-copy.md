## 소개 섹션

**타이틀:**
> b.stage로 무엇을 만들 수 있나요?

**본문:**
> b.stage는 250개 이상의 글로벌 클라이언트, 800개 이상의 아티스트가 사용하는 팬덤 플랫폼 빌더입니다. 개발자 API를 통해 여러분만의 브랜드 팬덤 플랫폼을 설계하고, b.stage의 인프라 위에서 독자적인 서비스를 운영하세요. 팬 데이터와 콘텐츠의 **완전한 소유권**은 항상 여러분에게 있습니다.

## 핵심 기능 섹션 (카드형)

**타이틀:** `핵심 API 기능`

### 1. 커뮤니티 & 콘텐츠
> **Fan Community API**
> 팬 피드, 게시판, Star Story 등 커뮤니티 기능을 내 서비스에 통합하세요. 독점 콘텐츠 관리와 설문, 투표 기능까지 API 한 번으로 구현됩니다.

### 2. 실시간 소통
> **b.stage POP API**
> 라이브 스트리밍, 1:1 DM, Q&A — 아티스트와 팬이 실시간으로 연결되는 경험을 직접 빌드하세요.

### 3. 멤버십
> **Membership API**
> 티어별 유료 멤버십, 전용 콘텐츠 접근 제어, NFT 기반 멤버십 게이팅까지. 지속 가능한 팬덤 수익 모델을 설계합니다.

### 4. 커머스
> **b.stage SHOP API**
> 실물 굿즈, 디지털 상품, 티켓, 바우처를 전 세계에 판매하세요. 230개국 글로벌 결제 및 배송 인프라를 그대로 활용할 수 있습니다.

### 5. 데이터 & 인사이트
> **Datalabs API**
> 팬 행동 데이터와 비즈니스 인사이트를 실시간으로 조회하세요. 데이터의 100% 소유권은 항상 여러분에게 있습니다.

### 6. Web3 / 블록체인
> **Web3 Integration API**
> Ethereum, Klaytn 기반 NFT 인증, 소셜 토큰, 크립토 지갑 연동을 손쉽게 구현하세요.

## 시작하기 섹션

**타이틀:** `3단계로 시작하는 b.stage 개발`

1. **계정 생성 & API 키 발급**
   > 개발자 계정을 만들고 API 키를 발급받으세요. 샌드박스 환경에서 무료로 테스트할 수 있습니다.

2. **문서 읽기 & SDK 설치**
   > REST API 레퍼런스와 각 기능별 가이드를 확인하고, 원하는 언어의 SDK를 설치하세요.

3. **빌드 & 배포**
   > b.stage의 인프라 위에서 나만의 팬덤 플랫폼을 배포하세요. 스케일업은 b.stage가 책임집니다.
